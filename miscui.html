BIG NOISE - Global Game Jam 2018 (Unity)

Frog Fractions - Game of the Decade Edition (2020) - "Presidential Bugberry"


BIG NOISE was my Global Game Jam 2018 entry, a fun little dance game with a "factory pomo" aesthetic. I still think it looks pretty good, so in my portfolio it remains. You can download and play BIG NOISE (on a Windows PC) from this link.

Along with many other talented developers, I was graced with a cameo in the hidden story mode for Frog Fractions - Game of the Decade Edition in 2020. I was responsible for Hop's "Presidential Bugberry", which served as the in-game conversation log. This photorealistic mobile device was rendered entirely in Photoshop!
