CROSSNIQ+

October 2019
Nintendo Switch/PC/Mac/Linux
Unity


Based on the original CROSSNIQ browser game I released in 2017, CROSSNIQ+ was my first console game and was entirely solo developed, including Switch porting.

CROSSNIQ+ expanded upon the original game's mechanics with new modes, including a multiplayer battle mode, and utilizes Switch features like the touch screen and HD Rumble.

For trailers, footage, and more screenshots, check out the CROSSNIQ+ Presskit() here.

I'm a game developer, graphic designer, and multimedia artist based in Cleveland, OH - but I could be coming to your town, if you'd like.

I graduated from DePaul University's CDM in 2015 with a BS in Video Game Development. Since then, I've created VR/AR installations, web games, narrative games, and even made the jump to full-time indie developer for my first console title, CROSSNIQ+ (Switch).

For my next adventure, I'm looking to hang up the indie mantle and work in a game dev environment where I can grow, collaborate, and ultimately help build something great. If you know a place like that, we should talk!
